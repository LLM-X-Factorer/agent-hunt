## 配文

分析了 9,287 条 AI 岗位 JD（国内 3,786 + 海外 5,501），想看看国内外 AI 招聘到底差在哪。

发现差的不是水平，是**招聘的语言**。

📊 国内 AI 岗位技能 top 5：
1. LLM（6.1%）
2. Agent Architecture（2.4%）
3. Python（1.6%）
4. RAG（1.0%）
5. Prompt Engineering（0.9%）

📊 海外 AI 岗位技能 top 5：
1. Python（4.2%）
2. LLM（2.5%）
3. Generative AI（2.3%）
4. SQL（1.8%）
5. Machine Learning（1.7%）

国内前 5 全是「调模型的工具」（LLM / Agent / RAG / Prompt）。
海外前 5 是「工程基础 + AI 概念」（Python / SQL / ML / GenAI）。

更扎心的是 top 10——海外把 SQL / AWS / PyTorch / ML 都写明，国内 JD 一个都没进 top 10。

不是国内程序员不会，是国内 JD 不写。

📌 国内招"会用大模型的人"。
📌 海外招"能建 AI 系统的人"。

如果你是国内 AI 工程师想去海外，简历要把这 5 个写到最前面：
1. Python（生产级工程代码，不是写脚本）
2. SQL + 数据分析（很多 AI 岗本质是数据 + 模型混合）
3. ML 基础（训练 / 评估 / 调优思路）
4. 至少一个云（AWS 优先）
5. Generative AI / RAG 实战经验

国内 JD 不写的东西，恰好是海外 JD 必写的。
