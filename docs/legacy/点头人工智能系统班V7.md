## Phase 3：AI应用实战——从模型到产品

### 阶段目标

通过行业场景驱动的项目实战，掌握 LLM/Agent 工程基础与模型部署方法，结合科研阶段已学的视觉算法，运用 Vibe Coding 工作方式，具备独立完成 AI+行业应用原型的能力。

- 掌握 LLM API 调用与 Prompt Engineering 系统化方法；
- 理解 RAG 与 Agent 工作流的核心概念并能应用于实际场景；
- 在行业场景中完成「视觉模型 + LLM」的联合应用；
- 使用 Vibe Coding 方式独立完成一个可运行的 AI 应用 Demo 并部署交付。

### 设计理念

科研阶段的核心方法论——拆解问题、设计实验、验证、迭代——同样适用于工程化场景。本阶段不是另起炉灶，而是将科研阶段建立的能力迁移到产品开发中：

| 科研阶段已学 | 应用阶段对应 |
|------------|------------|
| 模型选择与对比实验 | 选什么 LLM、怎么评估效果 |
| 数据预处理与增强 | 数据清洗、向量化、知识库构建 |
| 模型改进与消融实验 | Prompt 优化、RAG 调优、Agent 工作流设计 |
| 实验报告撰写 | 项目文档、README、Demo 演示 |
