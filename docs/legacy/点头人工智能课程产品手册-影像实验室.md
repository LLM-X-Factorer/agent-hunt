## 前言 Foreword

从V1.0到第六期，我们经历了一次又一次的升级与完善：

我们的课程一直在不断迭代，在前四个版本中，课程核心聚焦在"讲算法"，帮助学生快速入门深度学习。但随着人工智能技术的飞速发展，市面上涌现出越来越多关于算法的课程，而学员对于AI的能力需求也在不断提高，单纯的算法课程学习已经不能满足学员在科研上的需求。于是从V5.0开始，我们将课程定位为"基于数据与模型方法的AI科研入门解决方案"。

## 二、实验室介绍 Laboratory Introduction

### 2.1 实验室理论

在教学实践中，我们发现人工智能的学习路径与数据类型高度相关。通用课程难以应对各细分领域的独特需求。

为此，点头人工智能课程第六期按数据维度划分了三大实验室：**影像数据**、**时序数据**与**AI+科学数据**。每个实验室均配备定制化的课程与实战任务，聚焦于其核心数据特性与应用场景，旨在帮助学员在特定领域内实现深度掌握。

### 2.2 影像实验室研究与应用方向

#### 研究方向

影像数据实验室（Image Data Laboratory）专注于图像与视频类数据的智能处理，其中所涉及的影像数据，是以图像或视频形式呈现为信息载体的一切以像素矩阵形式存在的数据，即像素之间的位置关系，计算机通过识别这些规律来"看懂"图像。

本实验室所涉及的影像数据，按维度与形态可划分为：

**一维图像**
为便于视觉模型处理而将信号转化成的图像，例如：心电图（ECG）、脑电图（EEG）、频谱图、声谱图。

**二维图像**
单帧图像，例如：自然场景照片、医学影像（X光片、CT切片）、遥感卫星图像、工业检测图像、人脸识别图片、扫描文档与手写笔记、艺术绘画与设计稿。

**三维图像**
包含时间或空间深度的数据，例如：视频文件、视频帧序列、监控视频、激光雷达点云、三维重建模型、医学三维扫描（MRI）。

影像数据实验室以这些不同形式的影像数据为核心对象，开展针对性的算法训练与模型优化，帮助学员掌握图像处理、目标识别、视频分析等关键技术，推动科研与项目落地。

#### 应用方向

基于上述数据类型的分析与处理技术，主要核心场景应用于以下领域：

| 应用领域 | 说明 |
|---------|------|
| **计算机视觉（CV）** | 图像分类、目标检测、语义分割等 |
| **医学分析** | 疾病影像筛查、病灶自动分割与定位 |
| **工业视觉** | 产品表面缺陷检测、零件自动化质检 |
| **自动驾驶** | 车辆与行人感知、可行驶区域分割、交通场景理解 |
| **遥感观测** | 地表覆盖分类、自然灾害评估与环境变化检测 |
| **安防监控** | 人脸识别、人体行为分析、人群密度估计 |
| **内容生成** | 图像/视频的生成、编辑与风格迁移 |

### 2.3 影像实验室课程

#### 课程特色

影像数据实验室专为从事图像与视频数据处理方向的学员打造，课程体系与服务内容聚焦实践驱动与能力提升，主要体现在以下四方面：

**专属"三段式"进阶体系**
课程分为新手通识、AI科研、前沿技术三阶段，帮助学员从零基础掌握影像类算法与模型优化方法，最终具备独立完成个人影像数据项目的综合能力。

**项目驱动的多维影像任务训练**
围绕一维、二维及视频等多类影像数据，设计贴合实际的科研与工作任务，实现知识向能力的高效转化，支持个性化研究方向发展。

**能力提升路径清晰可量化**
围绕影像数据处理的关键环节，系统培养三大核心能力：模型调优、创新迭代、数据迁移。通过任务成果与阶段评估，实现科研能力的可视化成长。

**五位一体的导师团队**
实行问题清零责任制，为学员提供多方面的支持。在课程学习、技术实践和项目研发等环节，学员均可获得相应的帮助。学习过程中遇到的问题，可以通过指导、答疑和讨论等方式得到解答。

#### 课程项目

**CNN实现猫狗分类**

- **项目背景：** 图像分类是计算机视觉的基础任务，猫狗分类作为经典案例，适合初学者理解CNN原理及实战流程。
- **项目要求：** 构建CNN模型，使用包含猫狗的图像数据集进行训练，实现高精度分类，并完成模型部署测试。
- **技术介绍：** 采用卷积层、池化层和全连接层设计CNN网络；使用ReLU激活函数、Dropout防止过拟合；通过数据增强提升泛化能力；基于TensorFlow/PyTorch实现。

**ViT图像猫狗分类项目**

- **项目背景：** Vision Transformer（ViT）将自然语言处理的Transformer结构引入视觉领域，突破传统CNN局限，探索前沿技术应用。
- **项目要求：** 利用ViT架构完成猫狗图像分类，对比CNN模型效果，分析其优势与不足。
- **技术介绍：** 将图像分割为序列块输入Transformer；使用多头自注意力机制提取特征；加载预训练模型进行迁移学习；基于PyTorch或Hugging Face库实现。

**U-Net物品分割项目**

- **项目背景：** 图像分割在医疗、自动驾驶等领域应用广泛，U-Net凭借编码器-解码器结构在分割任务中表现优异。
- **项目要求：** 实现U-Net模型，对特定目标（如人物、物品）进行像素级分割，输出精准掩码。
- **技术介绍：** 设计对称编码器-解码器结构；使用跳跃连接融合浅层与深层特征；采用Dice损失函数解决类别不平衡；基于TensorFlow/Keras搭建模型。

**YOLO目标检测项目**

- **项目背景：** 目标检测需同时定位与识别物体，YOLO系列算法以"一步检测"高效性成为工业界热门选择。
- **项目要求：** 使用YOLOv8经典算法版本训练模型，检测图像或视频中的多类目标，并优化检测速度与精度。
- **技术介绍：** 利用Anchor机制预测边界框；采用CIoU损失优化回归精度；使用Mosaic数据增强；基于Ultralytics库快速开发与部署。

**基于CLIP的相似图像检索**

- **项目背景：** CLIP模型通过图文对比学习实现跨模态理解，为图像检索提供无需标注的新范式。
- **项目要求：** 利用CLIP提取图像与文本特征，构建检索系统，实现输入文本或图像查询相似内容。
- **技术介绍：** 调用预训练CLIP模型编码图像与文本；计算余弦相似度匹配特征；使用Faiss库加速向量检索；基于OpenCLIP库微调领域数据。

**强化学习量化投资交易**

- **项目背景：** 强化学习可通过与环境交互学习最优策略，适用于量化交易中的动态决策问题。
- **项目要求：** 构建RL智能体，学习股票或期货交易策略，最大化累积收益并控制风险。
- **技术介绍：** 使用DQN或PPO算法设计奖励函数；处理金融时间序列数据；引入风险惩罚机制；基于Stable-Baselines3或回测平台验证策略。
