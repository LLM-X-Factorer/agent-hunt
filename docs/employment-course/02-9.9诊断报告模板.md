# 9.9 诊断报告 — 内容模板与设计规范

> 漏斗入口的核心交付物。**报告主体不推课，最后单独一节"3 种路径"诚实推荐免费资源**——这是诚信路线最硬的证据，也是最容易被截图传播的部分。
>
> 双形态：PDF（5-10 页，传播+截图）+ Online Dashboard（互动探索）

---

## 0. 设计原则

1. **数据优先于结论**：每一个判断都附带 Agent Hunt 数据来源（X 条 JD 反推），不空喊
2. **诚实推免费资源**：报告里列真正有用的免费学习资源，不为了显得"良心"故意写差
3. **不催买课**：报告主体只讲"你是谁、市场要什么、怎么补"。课程只在最后一节作为 3 选 1
4. **可截图、可传播**：每一页都设计成能独立截图发朋友圈/社群，传播即获客
5. **透明披露生成依据**：附录注明"基于 X 条 JD、X 个角色聚类、SCI 评分模型生成"，反衬竞品的"凭空话术"

---

## 1. 用户输入表单（生成报告的基础）

支付前/支付后让用户填的表单。建议在支付完成后才填（提高 9.9 转化率），填完才生成报告。

| 字段 | 类型 | 必填 | 用途 |
|---|---|---|---|
| 当前职位 + 行业 | 文本 | ✅ | 推断转岗起点 |
| 工作年限 | 选项（应届/1-3/3-5/5-10/10+） | ✅ | 薪资定位依据 |
| 学历 | 选项（专科/本科/硕士/博士） | ✅ | 部分 JD 学历过滤 |
| 已掌握技能 | 多选标签 + 自定义 | ✅ | 匹配度计算核心 |
| 目标 AI 方向 | 多选 4 主线 + "我不知道" | ✅ | 推荐 Top 3 角色锚点 |
| 转岗动机 | 单选（升职/逃离 35 危机/兴趣/副业/其他） | 推荐 | 文案个性化 |
| 行业偏好 | 多选（互联网/金融/医疗/制造/...） | 推荐 | 角色推荐过滤 |
| 期望薪资范围 | 数字区间 | 推荐 | 薪资 reality check |
| 时间预算 | 选项（每周 5h/10h/20h+） | 推荐 | 学习路径推荐 |
| 城市 | 文本 | 推荐 | JD 地理过滤 |

**表单设计原则**：5 个必填 + 5 个推荐填，10 项 3 分钟内填完。不要超过 15 项。

## 2. 报告 7 节内容详情

### 第 1 节 · 封面 + 你是谁速览（1 页）

**内容**：
- 标题："{用户名}的 AI 求职定位报告"（个性化）
- 速览卡片：
  - 你的当前位置：{职位} · {行业} · {经验}年 · {学历}
  - 你的 AI 转岗目标：{4 主线匹配度图，bar 横向}
  - 你最匹配的 3 个角色：{角色名 + 匹配度 %}
  - 报告生成时间 + 报告 ID（可分享链接）

**数据来源**：用户输入 + 角色匹配算法

**视觉**：第 1 页是"传播页"，必须设计成可截图发朋友圈

---

### 第 2 节 · 市场全景：14 角色 × 你的位置（2 页）

**内容**：
- 国内 14 个 AI 角色全景图（横向气泡图：JD 数 × 中位薪资 × 学历门槛）
- 你的 Top 3 角色高亮 + 配文："基于你的 {经验/技能/目标}，你和这 3 个角色匹配度最高"
- 每个 Top 3 角色卡片：
  - JD 数（"国内市场近期共 {N} 条 JD 招聘此角色"）
  - 中位薪资 + P25/P75 区间
  - 典型公司（用 Agent Hunt 数据脱敏后展示，"中型互联网公司 + 头部金融科技 + 制造业 AI 转型公司"）
  - "为什么和你匹配"（基于用户技能 vs 角色 required_skills 的重合度，给 1 句话理由）

**数据来源**：`frontend/public/data/roles-domestic.json`

**视觉**：气泡图 + 3 张角色卡片

---

### 第 3 节 · 薪资定位 reality check（1-2 页）

**内容**：
- 散点图：行业经验 (X) × 月薪 (Y)，背景是市场全部 JD，红点是你的预期落点
- 配文：
  - "目标角色 {Top 1} 的薪资分布：P25={X} / P50={Y} / P75={Z}"
  - "按你的 {经验}年 经验，市场中位薪资约为 {N}"
  - 如果用户期望 > P75：友善提示"你的预期偏向 Top 25%，达成需要..."
  - 如果用户期望 < P25：提示"你的预期偏低，市场实际可以更高"
- **不打鸡血也不泼冷水**，纯数据陈述

**数据来源**：`salary-by-experience.json` + `roles-domestic.json`

---

### 第 4 节 · 技能 Gap 分析（1-2 页）

**内容**：
- 表格：你的目标角色 Top 1 的所有 required_skills + preferred_skills
- 每个技能 3 列标记：
  - 你已掌握 ✅ / 部分掌握 🟡 / 没掌握 ❌
  - **SCI 关键性分数**（这个技能在该角色里出现频率 + 重要性）
  - 优先级建议：先补 / 后补 / 锦上添花
- 配文：
  - "你已掌握 {N}/{M}，gap 主要在 {3 个核心技能}"
  - "按 SCI 排序，建议你先补 {Top 3 技能}"
  - 列出每个 gap 技能的具体学习起点（链接到第 5 节路径）

**数据来源**：`skill-gaps.json` + 角色 skill data + SCI 评分模型

---

### 第 5 节 · 3 种学习路径（**核心节，诚实推荐**）（2-3 页）

⚠️ **这是报告里最容易被传播的一节**。每条路径必须真实有用，不为了卖课故意贬低自学路径。

#### 路径 A：自学（适合时间充裕 + 自驱力强）

**适合谁**：
- 每周可投入 10h+
- 历史上完成过自学项目（不是仅"想学"）
- 经济敏感

**3 个月学习路线 + 真实免费资源**：

| 月份 | 学什么 | 推荐资源 | 投入时间 |
|---|---|---|---|
| 第 1 月 · 通识 | LLM 基础 / Prompt / Agent 概念 | • Datawhale 公益训练营（datawhale.club）<br>• 阿里云 ModelScope 训练营（modelscope.cn）<br>• 李宏毅 LLM 系列（B 站）<br>• Anthropic Prompt Engineering Guide（官网，免费英文） | 30-40h |
| 第 2 月 · 工具实战 | Cursor / Claude Code / Dify / Coze | • Cursor 官方教程<br>• Dify 官方文档 + Coze 模板市场<br>• 即刻"AI Agent 玩家"社区帖子<br>• YC AI Cookbook（cookbook.openai.com） | 30-40h |
| 第 3 月 · 主线深化 + 项目 | 按你目标主线深化（4 选 1） | **A 主线 AI PM**：人人都是产品经理「AI 产品经理」专题 + 派分享会<br>**B 主线 AI 运营**：秋叶 AI 办公训练营（1599，付费但便宜）+ AI 产品榜运营案例库<br>**C 主线 AI 转型咨询**：麦肯锡/Gartner/德勤 AI 报告 + 清华 AI 应用案例库<br>**D 主线 AIGC 创意**：B 站 ComfyUI 教程 + Liblib 创作社区 + Civitai | 40-60h |

**自学风险**：80% 自学者中途放弃；缺反馈循环、孤单。

#### 路径 B：3800 就业班（适合需要节奏感 + 教练 + 同伴）

**适合谁**：
- 有完整 12 周时间窗口
- 需要外力推动（自己不太能逼自己）
- 需要有人帮你判断"我做得对不对"
- 想要可以写进简历的项目作品

**你会得到**：
- 12 周陪跑（每周 1 次群直播 + 1 次 1V1 教练 + 班主任督学）
- 一个真实项目（按你主线方向定制）
- 完整 portfolio（简历改造 + Demo 视频 + 模拟面试录像）
- 加入校友社群

**透明承诺**：
- ❌ 不承诺包就业（业内"包就业"已被监管教育过）
- ✅ 承诺毕业 90 天后公布全班真实就业数据（offer 率/薪资/失败案例）
- ✅ 不挂续费班

**报名时机**：每月开放 1 次招生，下一期 {日期}。**不打饥饿营销**。

#### 路径 C：1V1 深度服务（999+，规划中）

**适合谁**：
- 已经有清晰目标（不需要诊断）
- 需要 1V1 个性化指导（求职策略 / 简历改造 / 面试模拟）
- 时间稀缺（陪跑班的 12 周用不上）

预计 2026 年底上线。当前阶段建议先选 A 或 B。

---

### 第 6 节 · 7/30/90 天 Micro-Action（1 页）

**内容**：不绑定任何路径，纯执行建议。

| 时间 | 1-2 件具体的事 |
|---|---|
| **未来 7 天** | 1. 用 Cursor / Claude Code 完成你日常工作中的 1 个小自动化（哪怕只是写邮件脚本）<br>2. 加入 1 个 AI 学习社群（Datawhale / 即刻 AI Agent 玩家 / 任选） |
| **未来 30 天** | 1. 完成 Datawhale 一期公益训练营（免费）<br>2. 在 Dify 或 Coze 上搭一个能跑的 Agent，记录过程 |
| **未来 90 天** | 1. 在 GitHub 或个人主页上展示你的第一个 AI 项目<br>2. 重新做一次本诊断报告，对比你的 Gap 缩小了多少 |

**关键设计点**：90 天后让用户"重新做一次诊断"是天然的复访钩子，不需要额外营销。

---

### 第 7 节 · 附录：报告生成依据 + 透明披露（1 页）

**内容**：
- "本报告基于 Agent Hunt 平台 {X} 条真实招聘 JD 生成"
- "数据更新至 {日期}，下次更新 {日期}"
- "国内 14 + 海外 11 角色聚类来自我们的开源分析项目（GitHub: ...）"
- "SCI（Skill Criticality Index）评分模型说明：..."
- "我们承诺不会向第三方出售你的数据；你的报告仅用于个人查阅"

**关键设计点**：附录是"诚信资本"，让懂行的人能验证报告依据，反衬竞品的"凭空话术"。

## 4. PDF vs Dashboard 形态差异

| 维度 | PDF（生成时锁定） | Dashboard（动态可探索） |
|---|---|---|
| 页数 | 5-10 页 | 单页应用，可滚动 |
| 第 2 节市场全景 | 静态气泡图 + 你的高亮 | 互动气泡图（hover 看每个角色详情、可切换"含/不含 你的目标主线"） |
| 第 4 节技能 Gap | 表格 | 互动表格（按 SCI 排序、可点击每个技能看相关 JD 样本） |
| 第 5 节路径 | 完整列出 3 路径 | 同上（不删减） |
| 用户复访 | 报告 ID 可生成新 PDF | 链接永久有效，数据每月自动更新（让用户感觉"每月都在涨知识"） |
| 分享 | 整篇 PDF 下载 | URL 分享，浏览器即看 |
| 截图传播 | 第 1、5 节封面页天然适合截图 | 同样可截图，但不及 PDF 完整 |

**双形态实现优先级**：
1. 先实现 PDF（用 Python WeasyPrint 或 React-PDF 渲染）
2. Dashboard 复用现有 Recharts 前端，加路由 `/diagnosis/{report_id}`
3. PDF 和 Dashboard 共享同一份 JSON 数据，避免双重维护

## 6. 实现优先级（落地路线）

| 优先级 | 任务 |
|---|---|
| P0（首版，必须做） | 用户输入表单 + 角色匹配算法 + 第 1/2/4/5/7 节静态生成 + PDF 输出 |

## 附录 · 报告生成的技术栈选型建议

- **后端生成**：Python + Jinja2 模板 + WeasyPrint（HTML → PDF）
- **数据查询**：复用现有 Agent Hunt FastAPI 端点
- **Dashboard 前端**：Next.js（已有）+ Recharts（已有）+ 新路由 `/diagnosis/{id}`
- **报告存储**：S3 / 阿里云 OSS（PDF 永久存档 + URL）
- **支付集成**：微信/支付宝小程序支付，9.9 元低客单适合移动端入口
