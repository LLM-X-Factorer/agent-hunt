## 📝 文档约定

- **新增就业班相关文档** → `employment-course/`，按 `XX-主题.md` 命名（XX 是序号）
- **新增平台技术文档** → `agent-hunt/`
- **临时报告/调研** → 同样放 `employment-course/`，文件名带日期
- **不要直接往 `docs/` 根目录放新文档**（除了本 README）
- 中文文件名 OK（保持已有风格），目录名用英文
